# Landing Page Redesign — 既存デザインを参考に作り直し

## Task
~/bg-growth-site/ のランディングページ（index.html, style.css, sample.html, sample.css）を作り直してください。

## 手順

### ステップ1: 参考デザインを読み込む
まず以下のファイルを読んでください:
/Users/suuuuui/ai-growth-landing/index.html

このファイルのデザイン（カラーパレット、フォント、レイアウト、アニメーション、全体的な雰囲気）を参考にしてください。

### ステップ2: 参考デザインの分析
読み込んだindex.htmlから以下を抽出:
- カラーパレット（背景、テキスト、アクセント、カード等）
- フォントファミリーとサイズ
- レイアウトパターン（セクション構成、カード配置、グリッド）
- アニメーション（ホバーエフェクト、スクロールアニメーション等）
- 全体的なデザインのトーン（高級感、暖かみ、知的さ等）

### ステップ3: BG Growthランディングページに適用
参考デザインのスタイルを、BG Growthのランディングページの**コンテンツはそのまま維持**しつつ適用してください。

変更するもの:
- style.css — カラーパレット、フォント、レイアウト、アニメーションを参考デザインに合わせて全面書き換え
- index.html — 必要に応じてHTML構造を調整（クラス名変更等）。ただしテキスト内容は変えない
- sample.css — sample.htmlのスタイルも統一
- sample.html — 必要に応じて構造を調整

変更しないもの:
- テキスト内容（コピー、見出し、説明文）
- ページのセクション構成と順番
- script.js のfade-inアニメーション機能
- レスポンシブ対応（モバイル対応を維持）

### ステップ4: 品質チェック
- ブラウザで開いて表示を確認
- モバイル幅（375px）でも崩れないか確認
- 全セクションが参考デザインのトーンと統一されているか確認

### ステップ5: デプロイ
git add -A && git commit -m "redesign landing page based on dark academia reference" && git push
